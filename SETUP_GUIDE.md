# Interview Timer Cards - セットアップガイド

## 🚀 クイックスタート

### 必要な環境
- Node.js 18.0以上
- npm または yarn

### インストール手順

1. **依存関係のインストール**
```bash
cd interview-timer-cards
npm install
```

2. **開発サーバーの起動**
```bash
npm run dev
```

3. **ブラウザでアクセス**
```
http://localhost:3000
```

## 📱 PWAとしてインストール

### スマートフォン（iOS）
1. Safariでアプリを開く
2. 共有ボタン（四角に上矢印）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### スマートフォン（Android）
1. Chromeでアプリを開く
2. 右上のメニュー（︙）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### デスクトップ（Chrome/Edge）
1. アドレスバーの右側にある「インストール」アイコンをクリック
2. 「インストール」をクリック

## 🎯 使い方

### 1. ランダムモードで練習
- ホーム画面から「ランダムモード」をタップ
- デッキを選択
- 録音するかどうかを選択
- 「開始」をタップして練習開始

### 2. 本番モードで全問練習
- ホーム画面から「本番モード」をタップ
- デッキを選択
- 全ての質問を順番に練習

### 3. 自分の質問を追加
- 「質問管理」から新しいデッキを作成
- 質問を追加（質問文と推奨秒数を入力）
- CSV形式でのインポート/エクスポートも可能

### 4. 設定のカスタマイズ
- 「設定」から以下を調整可能：
  - 準備時間の有効化・秒数
  - 本番モードの並び順
  - デフォルト回答時間
  - 録音の保存上限

### 5. 練習記録の確認
- 「履歴」から過去の練習を確認
- 苦手な質問をランキング表示
- 録音を再生して振り返り

## 💡 Tips

### 録音機能を活用する
- 自分の回答を録音して後から聞き返すことで改善点が見つかります
- マイクの許可が必要です（初回のみ）

### 準備時間を設定する
- 質問を読んで考える時間が欲しい場合は設定で有効化
- デフォルトは5秒（カスタマイズ可能）

### CSVで質問を一括管理
- Excelなどで質問リストを作成
- CSV形式で保存（形式: "質問文,推奨秒数"）
- 「質問管理」からインポート

### 苦手質問を重点的に練習
- 履歴画面の「苦手質問ランキング」を確認
- タイムオーバーが多い質問を重点的に練習

## 🔧 トラブルシューティング

### 録音ができない
- ブラウザのマイク許可を確認
- HTTPSまたはlocalhostでアクセスしているか確認

### データが消えた
- LocalStorageを使用しているため、ブラウザのキャッシュクリアで消える可能性があります
- 重要なデッキはCSVでエクスポートして保存することをお勧めします

### PWAがインストールできない
- HTTPSまたはlocalhostでアクセスしているか確認
- Service Workerが正しく登録されているか開発者ツールで確認

## 📊 データ管理

### データの保存場所
- すべてのデータはブラウザのLocalStorageに保存されます
- サーバーには一切送信されません（完全にローカル）

### データのバックアップ
1. 質問管理画面で各デッキをCSV形式でエクスポート
2. 履歴データは現在手動バックアップ機能なし（将来の機能追加予定）

### データの削除
- ブラウザの設定からサイトデータを削除
- またはブラウザキャッシュのクリア

## 🎨 カスタマイズ

### デフォルト質問の編集
- 「よくある面接質問」デッキは編集・削除不可
- カスタマイズしたい場合は新しいデッキを作成してコピー

### テーマの変更
- 現在はブルー系のテーマのみ
- ソースコードを編集することで色を変更可能（globals.css）
